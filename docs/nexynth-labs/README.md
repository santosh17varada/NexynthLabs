## Getting started

```bash
cd nexynthlabs-website
npm install
cp .env.example .env.local
npm run dev
```

Open [http://localhost:3000](http://localhost:3000). Admin CMS: [http://localhost:3000/admin/login](http://localhost:3000/admin/login).
